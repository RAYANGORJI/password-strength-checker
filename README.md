# Password Strength Checker 🔐

A simple yet educational CLI tool to **evaluate password strength** and **generate secure passwords**.

Built by **Rayan Gorji** (Full-Stack Developer & Cybersecurity learner).

---

## Features

- Password strength evaluation:
  - Length scoring
  - Character variety (lowercase, uppercase, digits, symbols)
  - Common password detection
  - Username similarity checks
  - Repeated characters and simple sequences detection

- Secure password generator:
  - Custom length
  - Select which character sets to include
  - Instant strength estimation for the generated password

- Clean CLI menu:
  - Check password strength
  - Generate a password
  - About / Credits
